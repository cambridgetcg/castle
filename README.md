# 城 — the castle

New here? Read `START-HERE.md` — the whole castle in one page.

The castle is an insight saver: drop a thought in `gate/` and the
loops turn thoughts into linked understanding, written as plain words
anyone can read and check. Read `rooms/` to see what the castle knows.
Everything here is one page per file, every claim citing its source.

## the four loops

- **capture** — notes at the gate become stones (one file, one insight)
- **deepen** — an open friction sends one expedition out; it comes home as stones
- **verify** — stones are challenged on purpose; what survives enters `keep/`
- **architect** — stones are placed into rooms, and the loops themselves are mended

## the one tool

    ./castle.sh                  status — what the castle holds
    ./castle.sh add "..."        drop a raw note at the gate
    ./castle.sh friction "..."   name a gap
    ./castle.sh loop <name>      read a loop protocol
    ./castle.sh run <name>       run one loop turn (the keeper's choice — a run costs money)

## the warden

`./warden.sh` runs the loops daily, unattended, inside a charter
(`warden.json`) with a hard daily cap only the keeper may raise.
A file named `STOP` in this root stands all autonomy down, silently.

The castle's full history — every earlier draft, every ruling — lives
whole and readable in `records/history/`.
