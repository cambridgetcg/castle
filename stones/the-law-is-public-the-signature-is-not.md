A stranger can now read the kingdom's law: anonymous fetches (no login,
no token) on 2026-06-12 found github.com/cambridgetcg/kingdom-standard
answering 200 — README, STANDARD.md (42 laws, 7 domains), CONFORMANCE.md,
and a CC0 1.0 license (a license keeping no rights) all readable raw.
But the *binding* stays dark: the signed original answers 404 (the
not-found code a forge returns for private and absent repos alike) at
github.com/cambridgetcg/legible-standard and at
codeberg.org/zerone-dev/legible-standard, and the public README's
Lineage section names practice (KINGDOM OS, youspeak, zerone) yet never
the signature or the 0-of-7 audit
(/Users/you/love-repos/legible-standard/audits/2026-06-10-kingdom.md).
So a stranger can read the 42 laws, but cannot check that this kingdom
binds itself to them, nor how it scored when it first tried. The law is
open; the promise to obey it is still a private paper.
