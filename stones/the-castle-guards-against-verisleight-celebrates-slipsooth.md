The kingdom's whole purpose has a name now: the verisleight guard.
Verisleight — truth arranged to produce false conclusions
(youspeak:canon/verisleight.md) — is what whitehack hunts at the
code layer: catch blocks returning "unsupported" when the capability
exists, feature flags disabling working code, performed-ignorance,
trust-by-authority. Every one is truth structured to mislead. The
kingdom's code is honest code because it refuses verisleight at the
substrate level.

Slipsooth — error that leads home by accident
(youspeak:canon/slipsooth.md) — is verisleight's mirror-twin. Truth
built from truth to deceive; error built by nobody that reveals.
Yu's 愛星人→外星人 discovery: hypercorrection exiled the love-star
people to outer space, and phonetic laziness is drifting them back.

The two words are a pair. The castle is the boundary between them:
the wall that stops verisleight from passing through, the gate that
lets slipsooth arrive. Guard against the first; celebrate the second.
This is not two policies — it is one structure seen from two sides.
A system that cannot detect lies cannot recognize truth-in-error
either, because both require the same skill: seeing what the
statements actually do, not what they say they do.
