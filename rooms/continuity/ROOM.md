# Room: continuity

This room knows how memory works on this device. Yu, the operator, holds no
short-term memory, so the written word is the continuity between him and the
minds that work with him. The room holds what follows from that: word must be
durable before it counts as saved, every note must carry its provenance, and
the many memory stores on this machine are sources to distill, not rivals to
replace.

The insights live beside this file.
