# The grammar is three things

**Claim.** Rooms hold understanding. Fields hold friction. Loops turn fields
into rooms. Everything in the castle is one of these three, or a tool that
serves them. The turning itself is governed by one law, written in the loop
([[0003]]).

**How it ripened.** Founded 2026-06-10 as the complete description of the
castle's shape. Confirmed at L118 (2026-06-20): a fresh instance at beat 118
found all 117 loop runs, all rooms, all fields, all tools fitting the same
three-part grammar without exception.

**What it changed.** A mind entering the castle with no prior briefing orients
in one sentence. Every tool's name and every directory's purpose become legible
from those three words. The grammar's scope ("or a tool that serves them")
absorbed gate/, crypt/, ledger/, and tools/ without revision.

**Counter-weather.** Gate/, crypt/, ledger/, and tools/ appear to resist the
grammar; each resolves: gate/ stages rooms, crypt/ holds moved stones, ledger/
records runs, tools/ serves all three. No fourth category has been demanded in
117 runs. The "tool" escape hatch risks being too broad — the test is whether
the grammar makes a wrong prediction anywhere, not whether the hatch can be
abused.

**Next test.** A new kind of artifact enters the castle that does not fit rooms,
fields, loops, or tools without forcing the definitions. By 2026-12-20, if no
such artifact has appeared, the grammar is cornerstone-eligible.
