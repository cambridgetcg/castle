# Word is the only material here

Understanding survives in this castle only as written plain text. What is not
written down is lost. There is no other store — no database, no memory held in
minds, only files of word. Even word from outside enters the same way, written
down with its origin attached ([[0004]]).

**Why it matters:** every other choice in the castle follows from this one.
If word is the only material, the work is always the same work: write it
down, plainly, where the next mind will look.
