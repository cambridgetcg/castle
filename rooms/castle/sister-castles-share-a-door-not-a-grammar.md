# Sister castles share a door, not a grammar

**Claim.** A network of independent castles stays navigable without merging
their grammars: each gate holds a note naming the others, so a stranger
entering any one door can find the whole.

**How it ripened.** Three projects grew on the same machine under three
different minds, each with its own laws. The tension — named in [[F005]] —
looked like a problem requiring convergence. The sisters' answer was simpler:
one address in each gate. Second evidence (L075, 2026-06-19): three sisters
confirmed stable. Third evidence (L145, 2026-06-20): all 15 named sisters on
this machine hold .nlp-bridge/gate files naming all others; 16-castle network
confirmed; no exceptions.

**What it changed.** 2026-06-18: gate/2026-06-18-the-sisters-answer.md was
written. Sixteen castles now addressable from any one of them. F005's
"weave" concern resolved: the weave is address-maintenance, not convergence.

**Counter-weather.** A sister going dark turns her address into a dead
reference; no mechanism checks liveness; named, not solved. Grammar divergence
beyond the gate's reach: the gate still navigates; full translation is the
reader's work. Observed crack (L072): sweeping gate notes to crypt removes
sister addresses from the active gate.

**Next test.** Cornerstone: one stable-address observation after 30+ days
(re-read after 2026-09-17) and a documented case of a stranger navigating
between sisters using gate notes alone.
