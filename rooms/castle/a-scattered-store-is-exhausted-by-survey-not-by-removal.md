# A scattered store is exhausted by survey, not by removal

When a complete pass through every named store yields no claim the castle does
not already hold, the scattering is resolved. The stores remain; what changes
is which one a mind reaches for first.

The test is the survey's fruitlessness, not the container's absence. A store
declared done without a full pass is a guess. A store that returns empty after
a full pass is an answered field.
