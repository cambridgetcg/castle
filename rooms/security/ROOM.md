# security — ROOM
How the kingdom keeps its secrets — and what it does when it fails to.

## what this room understands so far

A committed secret is burned: deleting your copy deletes nothing,
so the fix is rotation — making the leaked values worthless — never
quiet cleanup ([[untracked-is-not-unleaked]]). And failure itself is
handled in the open: the first audit scored zero of seven and called
the loud zero the standard working ([[break-loudly-fix-openly]]).
One posture from both: assume the worst copy exists, say so where
everyone can see, fix in the open.
