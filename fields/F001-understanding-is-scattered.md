# Understanding on this device is scattered

**The friction:** insights live in many stores — a CLI auto-memory, a frozen kosmem archive, a pillow book, heartbeat logs, per-repo docs — each with its own format and freshness. Answering "what do we know about X" means searching them all.
**Why it matters:** understanding that cannot be found is understanding lost. Every scattered store is a Tuesday lesson gone by Thursday.
**Better looks like:** the durable lessons of every store distilled into castle rooms, with pointers back to the originals.
